# Keiba App (競馬スケジュール管理・馬券分析システム)

本アプリケーションは、競馬の開催スケジュール管理、出馬表および予想情報の閲覧、競走馬の詳細データ（血統・過去成績）の検索、ならびに馬券の収支分析を統合的に行うためのローカル環境向けWebアプリケーションです。

## 主な機能

- **レーススケジュール管理**
  - JRA公式情報および外部サイトを統合した開催スケジュールのカレンダー表示。
  - 各レースの詳細情報（出走馬、距離、馬場状態、予想印、オッズ、人気順等）の一覧表示。
  - システム未登録レースのユーザー手動追加、および不要なレースの削除・復元機能。
- **競走馬データ検索・分析**
  - ローカルデータとWebスクレイピングを併用した高速かつ網羅的な馬名検索（現役・引退馬対応）。
  - 血統情報（父・母・母父）、過去のレース成績の確認。
  - 人気・着順・馬体重の推移を可視化する折れ線グラフ機能。
  - 外部データベースからの引退・死亡日の自動抽出機能。
  - お気に入り馬の登録およびメモ機能。
- **馬券収支分析**
  - 購入馬券の掛け金、払戻金、収支、回収率、的中率の集計および履歴一覧表示。
  - 期間、馬場（芝・ダート）、馬券種別など、多様な条件による詳細な絞り込み機能。
  - 距離別、競馬場別、馬券種別などの切り口で分析可能なプリセットグラフ機能。
  - 任意のX軸・Y軸を設定可能なカスタムグラフ（棒グラフ・折れ線グラフ）作成機能。

## アーキテクチャおよび技術スタック

本システムはクライアント・サーバーモデルを採用し、外部サイトから取得したデータをローカルのJSONファイルとしてキャッシュ・管理する構成となっています。

### Frontend (`client/`)
- **React** (UI構築)
- **TypeScript** (静的型付け)
- **Tailwind CSS** (スタイリング)
- **Recharts** (グラフ描画)

### Backend (`server/`)
- **Node.js / Express** (APIサーバー)
- **TypeScript** (静的型付け)
- **Cheerio / Axios / iconv-lite** (データ収集・文字コード変換)
- **Local JSON Storage** (データ永続化およびキャッシュ管理)

## システムの処理フロー

1. **データリクエスト**: クライアントからカレンダー情報や競走馬の詳細についてAPIリクエストを送信。
2. **ローカルキャッシュの確認**: バックエンドにてローカル（`data/`ディレクトリ）のJSONファイルやメモリキャッシュを確認し、データが存在する場合は即座に返却。
3. **データ収集の実行**: データが存在しない、または更新が必要な場合、対象URL（JRA公式、netkeiba.com、Wikipedia等）へリクエストを送信。
4. **データの整形と保存**: 取得したHTMLをパースし、システムで扱いやすい形式に変換したのち、ローカルにJSONファイルとして保存。
5. **画面描画**: クライアントが受け取ったデータをもとに、ReactコンポーネントおよびRechartsを用いてUIを描画。

## 環境構築 (Setup)

### 前提条件
- Node.js (v18以上を推奨)
- npm または yarn

### インストール

リポジトリをクローン後、フロントエンドおよびバックエンドの各ディレクトリで依存パッケージをインストールします。

```bash
# リポジトリのクローン
git clone https://github.com/yourusername/keiba-app.git
cd keiba-app

# バックエンドのセットアップ
cd server
npm install

# フロントエンドのセットアップ
cd ../client
npm install
```

## 💻 実行方法 (Usage)

ローカル開発環境でサーバーとクライアントを起動します。
ターミナルを2つ開いて、それぞれ以下のコマンドを実行してください。

### 1. バックエンドサーバーの起動
```bash
cd server
npm run dev
```
※ デフォルトでは `http://localhost:3000` 等（環境による）でAPIサーバーが起動します。

### 2. フロントエンドの開発サーバーの起動
```bash
cd client
npm run dev
```
※ 起動後、ターミナルに表示されるURL（例: `http://localhost:5173`）にブラウザでアクセスしてください。
